# 浙政钉小程序开发上架流程



了解浙政钉之前，必须先了解专有钉钉



## 专有钉钉是什么？



​		首先要明确专有钉钉与浙政钉的区别，专有钉钉是一款Saas化的产品；类似原来的钉钉。通过公司的营业执照都可以申请注册。组织间的用户数据是租户隔离的。专有钉钉代码单独部署在浙江政务一朵云平台，也就是目前的浙政钉。因此严格意义上说专有钉钉和浙政钉没有任何关系。只是产品的功能基本一致。在专有钉钉可以独立管理自己的组织。管理自己的应用。最重要的是开放平台的接口是在线API，必须使用登陆账号才能查看相关接口说明。对于应用开发过程来说，专有钉钉是最好的测试学习平台。基于专有钉钉租户创建的组织、应用均可用于测试。方便后续对接浙政钉做准备。

## 三.常见问题



#### 		1.小程序模拟器上使用该api报错：[API_getAuthCode_error] 无效参数？



![img](./assets/zzd/2273512-20230306134118489-296210408.png)



​		目前模拟器暂不支持该api的调试，请使用IDE右上角的真机调试	



#### 		2.找不到设置白名单的地方

​		![image-20230801142710957](./assets/zzd/image-20230801142710957.png)



#### 		3.**为什么真机调试构建失败了**



​			可能没有配置白名单，或者登录失效（需重新扫码登陆开发工具，很坑）



####     	4.真机调试或者应用上架之后，手机上一直在loading或者转圈，或者直接空白



​			检查下小程序的首页路径是否为pages/index/index. 这个目前首页是不支持修改的

​			![image-20230801143549342](./assets/zzd/image-20230801143549342.png)



#### 		5.真机调试后或项目上线后，底部的tabBar导航不见了



![img](./assets/zzd/2273512-20230306141610260-1090324872.png)





#### 	 	6.不要给顶部导航栏设置背景色且专有钉钉小程序不支持自定义导航栏

​		

#### 	 	7.为什么扫码登陆后关联不了应用



​			不管什么环境，都需要应用管理员来扫码，专有钉钉环境可在专有钉钉门户的后台添加应用管理员。浙政钉环境只能去找浙政钉环境的应用管理员扫码了。

​	![img](./assets/zzd/2273512-20230306153108997-928770561.png)



#### 		8.例如富文本编辑小程序内部无法支持



​			如某些功能小程序内部不提供支持且是重点，可使用webView嵌套H5页面进行曲线救国



#### 	9.怎样获得租户ID

​				

##### 		（1）通过获取用户详情/rpc/oauth2/dingtalk_app_user.json这个接口返回的realmId即是租户ID。

##### 		（2）进入管理工作台，任意界面右击，查看网页源码，其中realmId即是租户ID。

​					![image-20230802090118609](./assets/zzd/image-20230802090118609.png)



#### 10.如果切换账号及环境，如果关联不到小程序，请重启小程序开发工具

#### 11.如果在页面初始化的时候调用dd相关Api

​		

```javascript
dd.ready(function() {
    //调用jsapi方法
})
```

​		如果 JSAPI 调用在页面初始化时就需要调用，则需要在 dd.ready 中进行调用。



#### 12.安卓13调用无法正常调用dd方法



​		一般专有钉钉会出现该问题，没有做兼容处理，直接反馈给技术支持（等浙政钉团队兼容即可）



#### 13.浙政钉版本升级，填写升级说明，需要说明要升级的版本号



#### 14.报错访问IP不在白名单，表示服务器出口IP没有配置对



​		可在服务器使用终端命令：curl ifconfig.me



#### 15.目前技术支持钉钉群组移到浙政钉群组，必须有浙政钉账号加群，进行技术支持沟通，每次沟通需要进行提交工单

## 四.浙政钉对比微信小程序



|                |                            浙政钉                            |                             微信                             |
| :------------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|    服务对象    |                             企业                             |                          个人，企业                          |
|  应用服务部署  |                           政务外网                           |                        自身服务器即可                        |
|    开发工具    | 支付宝小程序开发工具（选择mPaas类型，并且需要进行.json文件环境配置） |            微信小程序开发工具（无需进行环境配置）            |
| uni-app兼容性  |         兼容（但部分小程序内部不支持，如富文本编辑）         |    兼容（如直接获取手机本文件无法获取，只能获取聊天文件）    |
