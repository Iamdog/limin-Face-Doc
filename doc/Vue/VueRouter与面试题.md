# Vue Router 与面试题

## 知识点

- 路由模式：`hash` 与 `history`。
- 核心能力：动态路由、嵌套路由、路由懒加载、导航守卫。
- 常见守卫：全局前置 `beforeEach`、路由独享守卫、组件内守卫。

## 实战建议

- 页面组件用动态导入实现懒加载，降低首屏体积。
- 权限控制优先放全局守卫，统一处理登录态和角色权限。
- 404 与异常页要明确兜底路由。

## 高频面试题

### 1. `hash` 和 `history` 怎么选？
答：`history` URL 更美观，但服务端需配置回退；`hash` 对服务端要求低。

### 2. 导航守卫里可以请求接口吗？
答：可以，但要控制耗时并处理失败分支，避免路由阻塞体验过差。

### 3. 路由懒加载有什么价值？
答：减少初始包体积，提升首屏加载速度。
