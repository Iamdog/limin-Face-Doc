本章节按“单个知识点 + 面试题”拆分，方便系统复习和快速查漏补缺。

- 先理解核心机制：响应式、组件化、生命周期、路由、状态管理。

- 面试回答建议遵循：概念 -> 原理 -> 实战场景 -> 常见坑。
