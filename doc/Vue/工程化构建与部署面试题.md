# 工程化构建与部署面试题

## 知识点

- Vue 工程常见基建：Vite、ESLint、Prettier、TypeScript、单元测试。
- 构建优化关注：分包策略、按需加载、资源压缩、缓存命中。
- 部署阶段关注：静态资源版本化、CDN 缓存、回滚策略、监控告警。

## 实战建议

- 使用构建分析工具识别大依赖，优先按路由分包。
- 关键静态资源带内容哈希，避免缓存污染。
- CI 中增加质量门禁：lint、test、build 必须通过。

## 高频面试题

### 1. Vite 为什么启动快？
答：开发时基于原生 ESM 按需编译，减少传统打包器全量构建成本。

### 2. 如何处理发布后白屏问题？
答：先看静态资源 404 与缓存，再排查基础路径 `base`、路由回退和脚本错误。

### 3. 前端上线后如何快速回滚？
答：保留历史构建产物并配合网关/CDN 切换版本，确保可一键回退。
